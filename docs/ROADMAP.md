## Roadmap

### Phase 1: 增強記憶系統

| 項目             | 說明                 | 優先度 | 狀態 |
| ---------------- | -------------------- | ------ | ---- |
| 多對話上下文記憶 | 跨對話保持上下文連貫 | P0     | ⏳   |
| 任務歷史追蹤     | 記錄並摘要歷史任務   | P1     | ✅   |
| 跨 session 偏好  | 延續使用者偏好設定   | P1     | ⏳   |

### Phase 2: Party Mode

| 項目          | 說明                    | 優先度 |
| ------------- | ----------------------- | ------ |
| 多 Agent 並行 | 同時調用多個 Agent 協作 | P0     |
| 任務分片      | 自動拆分大型任務        | P1     |
| 狀態儀表板    | 即時查看所有 Agent 狀態 | P2     |

### Phase 3: 生態擴展

| 項目           | 說明                       | 優先度 |
| -------------- | -------------------------- | ------ |
| 更多領域 Agent | 設計/產品/測試等專業 Agent | P1     |
| 第三方技能市場 | 允許載入外部技能           | P2     |
| Web UI 控制台  | 圖形化控制面板             | P2     |

### Phase 4: 進階功能

| 項目       | 說明               | 優先度 |
| ---------- | ------------------ | ------ |
| 長期規劃   | 子目標分解與追蹤   | P2     |
| 主動式建議 | 主動預測使用者需求 | P2     |
| 多語言支援 | 支援英文等其他語言 | P3     |
